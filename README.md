# Wordle Game 

## Kör lokalt
```bash
npm install
npm start
# besök http://localhost:5080
```

## Struktur
- `server.js` – startar Express och kopplar routes
- `db/` – SQLite-init
- `routes/` – API och sidor (SSR + statisk)
- `services/` – ordlogik
- `views/` – SSR-mallar
- `public/` – klient (React via CDN)
